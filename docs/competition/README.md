## 核心创新点

### 1. MCP协议标准化数据接入
- 首次将MCP协议引入建筑能源管理领域
- 实现7个标准化工具接口(健康检查、能力声明、能耗查询、COP计算、异常检测、报表导出、统计汇总)
- 支持AI大模型通过自然语言直接调用能耗数据与统计功能
- 采用Streamable HTTP传输协议,支持远程访问与多客户端并发

### 2. LightRAG知识图谱驱动的智慧运维
- 基于知识图谱的双层检索机制(低层实体精确匹配+高层主题语义聚合)
- 自动解析运维手册、设备说明书、能耗数据字典等非结构化文档
- 支持local/global/hybrid/naive四种检索模式
- 实现跨文档信息整合与复杂问题推理

### 3. 结构感知路由多传感器异常检测
- 自适应学习传感器间依赖拓扑结构
- 通过路由机制实现异常传播分析与根因定位
- 双层检测架构(阈值检测+SAR-AD深度检测)
- 多尺度时间特征融合(分钟级到季节级)

## 系统特性

- **高性能**: 异步I/O架构,单接口响应时间<50ms
- **标准化**: MCP协议标准化数据接入,支持主流AI客户端
- **智能化**: LightRAG知识图谱+LLM,精准运维问答
- **可扩展**: 模块化设计,支持功能扩展与二次开发
- **开源**: MIT协议,代码完全开源

## 性能指标

- 核心查询接口平均响应时间: 42ms
- COP计算准确率: 100%
- 异常检测召回率: 89.3%
- 系统连续运行: 72小时无崩溃
- 支持数据规模: 210,000+条能耗记录

- 直接节能收益: 5%-15%节能效果
- 运维成本降低: 提升运维效率30%-40%
- 投资回收期: 2-3年
