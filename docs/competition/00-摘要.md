# 项目摘要

在"双碳"战略目标引领下，建筑行业作为碳排放核心源头之一，其运行阶段碳排放占全国总量的20%以上，亟需智能化能源管理手段实现绿色低碳转型。当前，建筑能源管理普遍面临运维依赖人工经验、知识传承困难、数据查询效率低、缺乏智能化分析手段等突出痛点。本项目围绕"能耗数据构建+查询统计+大模型智慧运维"一体化目标，研发建筑能源智能管理与运营优化系统（AERO），致力于为建筑能源管理提供智能化、一站式解决方案。

通过深入调研，我们总结出建筑能源管理领域四大核心痛点：（1）能耗数据分散、格式不统一，查询统计效率低下；（2）运维高度依赖人工经验，故障诊断响应慢、知识传承困难；（3）缺乏多维度智能化分析手段，能耗异常难以及时发现；（4）系统间数据孤岛严重，缺乏统一的数据交互标准与协议。针对上述痛点，本项目提出以下创新方案：（1）基于MCP（Model Context Protocol）协议构建标准化能耗数据接入与查询接口，支持按建筑、时间、监测参数等多条件精准查询，实现AI大模型与能耗数据的无缝对接；（2）融合LightRAG知识图谱增强检索技术，构建集成能耗数据字典、设备运维手册等领域知识库的智慧运维助手，支持自然语言问答、故障辅助诊断与运维流程指导；（3）引入基于结构感知路由的多传感器时间序列异常检测算法，实现能耗异常的智能化识别与诊断；（4）实现时段汇总、COP计算、数据异常分析等核心统计功能，支持多种可视化图表与自动化报表生成。

系统采用前后端分离架构，后端基于FastAPI异步框架与PostgreSQL时序数据存储，前端采用Vue 3 + Naive UI构建响应式交互界面。系统集成LightRAG知识图谱引擎，实现文档自动解析、实体关系抽取与多模式混合检索；通过FastMCP实现标准化MCP协议工具暴露，支持Claude Desktop等AI客户端直接调用能耗查询、COP计算、异常检测等核心功能。经测试，系统核心功能响应时间均达到演示级标准，数据集格式规范无明显异常，连续运行30分钟稳定无崩溃。
